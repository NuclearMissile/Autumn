# Autumn 
Yet another toy web application framework imitating Spring.

## Functions
- [x] IOC context
- [x] AOP injection
- [x] Jakarta EE servlet 
- [x] Web MVC framework
- [x] JdbcTemplate and simple ORM
- [x] Support .war standard 
- [ ] Self-bootable .war
- [ ] Demo webapp
- [ ] Homemade http server to replace Tomcat
